# Projectphp_webghostplace
# Description
The Haunted Places Finder system is a web application platform that allows users to search, share, and provide information about locations with mysterious stories or a history of paranormal activity across various areas.
# Scope of Work
User Management – Create, delete, and edit users
Haunted Location Management – Create, delete, and edit haunted locations
Location Image Management – Create, delete, and edit additional location photos
Review & Rating Management – Create, delete, and edit reviews and ratings
Review Vote Management – Create, delete, and edit review votes
